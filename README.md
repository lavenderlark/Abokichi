# **Abokichi**

## **About**

Abokichi is a responsive, multi-page e-commerce demo using pure HTML, CSS, and JavaScript. It features a clean green-accented design inspired by Japanese food branding.

---

## **Features**

* Multi-page navigation: Homepage, Products, Product Details, Cart, Checkout/Order Success
* Header with logo and cart badge on every page
* Hero section with gradient title and "Shop Now" button leading to products
* Product cards with name, image, price, rating, and add-to-cart
* Cart: add/remove items, change quantities, view total, checkout clears cart
* Filters: by category, type, flavor, and rating
* Sort options: price, rating, name
* Buy Now or Checkout places order and removes item(s) from cart
* Responsive layout (Flexbox & CSS Grid)
* Runs on plain HTML, CSS, JavaScript—no frameworks

---

## **Usage**

* Download or clone all files and open index.html in your browser.
* Browse categories or featured products on the homepage.
* Use filters or sorting to customize the product list.
* Add to Cart or Buy Now updates the cart; the cart can be managed from any page.
* Placing an order via Buy Now or Checkout shows the order success page.
* No installation or dependencies—everything runs in the browser.

---

## **Technologies**

* HTML5
* CSS3 (Flexbox & CSS Grid)
* JavaScript (arrays and DOM manipulation)
